# Jordi Design System | High-Fidelity UI Architecture

Este repositório contém o código-fonte de um sistema de design autoral, focado em romper com padrões genéricos de interfaces web e entregar uma estética premium "pixel-perfect".

## 🚀 Demo Online
Confira o projeto rodando em tempo real: [https://jordidesignsystem.netlify.app/](https://jordidesignsystem.netlify.app/)

## ✨ Diferenciais Técnicos
- **Tipografia Editorial:** Implementação de escalas fluidas e hierarquia visual avançada.
- **Micro-interações:** Lógica de hover e física de movimento estado-dependente via JavaScript Vanilla.
- **Performance:** Zero layout shifts e otimização de renderização com Tailwind CSS.
- **Glassmorphism:** Efeitos de profundidade e desfoque de fundo (backdrop-filter) modernos.

## 🛠 Stack
- HTML5 / CSS3 Moderno
- Tailwind CSS (Arquitetura de componentes)
- JavaScript Vanilla (Animações e interações)
